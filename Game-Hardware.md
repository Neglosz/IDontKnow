# Specification: IoT Circuit Sandbox Component (Reality Lens - NSC Project)

## 1. Objective & Game Context (เป้าหมายและบริบทของเกม)
เปลี่ยนระบบการทำข้อสอบปรนัย (Multiple Choice) ทฤษฎีฮาร์ดแวร์ IoT และระบบฝังตัว (Embedded System) แบบเดิม ๆ ให้กลายเป็นการแก้ปริศนาเชิงวิศวกรรมปลายเปิดที่ซ่อนการประเมินผลตามสภาพจริง (Stealth Assessment) 
- **ผู้เล่นสวมบทบาทเป็น:** "ช่างกลเวทมนตร์ประจำหมู่บ้าน"
- **สถานการณ์ด่านย่อย (Chapter 3):** ประตูโรงเรือนอัจฉริยะต้องการระบบเปิด-ปิดอัตโนมัติเมื่อมีชาวนาเดินมาถึง บอร์ดฝังตัว ESP32 หลักในฉากเกิดความเสียหาย ผู้เล่นต้องเลือกฮาร์ดแวร์เซนเซอร์ที่ถูกต้องจากคลังไอเทม นำมาติดตั้งและลากเส้นต่อสายไฟสัญญาณให้ระบบกลับมาทำงานได้สมบูรณ์

---

## 🎨 2. Visual Aesthetic & Theme (ธีมกราฟิกและดีไซน์ UI)
- **สไตล์หลัก:** 2D Retro Pixel-Art / 8-Bit Vintage Fantasy 
- **รูปทรงเรขาคณิต:** บังคับใช้กล่องขอบเหลี่ยมสันคม (Rectangular Geometry) **ห้ามใช้ขอบมน (Rounded Corners)** แบบแอปยุคใหม่เด็ดขาด
- **เส้นขอบและเงา:** ทุกองค์ประกอบต้องตัดขอบหนา 3px ด้วยสีน้ำตาลเข้ม (#2C1B10) และใส่ Inset Box Shadow เพื่อจำลองปุ่มและเมนูกระดานชนวนโบราณ
- **ชุดสี (Color Palette):**
  - หน้าจอหลักและแผงควบคุม: สีครีม/เบจสว่างอ่อน ๆ (#F5F2EB)
  - ม้วนกระดาษภารกิจ: สีเหลืองเก่าพาร์ชเมนท์ (#F7E7C4)
  - แท่นประกอบวงจรด้านหลัง: สีเหล็กเมทัลลิกเข้ม (#1E222A)
  - บอร์ดไมโครคอนโทรลเลอร์: สีเขียวลายวงจรชิป (#2E7D32)
- **ฟอนต์:** ใช้ Monospace Font ทั่วทั้งหน้าจอเพื่อคงความรู้สึกเทคโนโลยีเรโทร

## ⚙️ 3. System Architecture & State Management (สถาปัตยกรรมตัวแปรหลังบ้าน)
ส่วนประกอบ React Native (Expo) นี้ต้องควบคุมตัวแปรสถานะ (States) ดังต่อไปนี้เพื่อเชื่อมโยงกับ Core Game Loop:

```typescript
type SensorID = 'pir' | 'dht11' | 'soil' | 'ldr';

interface GameState {
  stars: number;          // คะแนนดาวสะสม (เริ่มต้นที่ 1200)
  combo: number;          // จำนวนการตอบถูกต่อเนื่อง (เริ่มต้นที่ 2)
  multiplier: number;     // ตัวคูณคะแนนปัจจุบัน (เริ่มต้นที่ 1.2)
  bestCombo: number;      // คอมโบสูงสุดที่เคยทำได้ (เริ่มต้นที่ 6)
  selectedSensor: SensorID | null; // ไอเทมที่ผู้เล่นคลิกเลือกในคลัง
  pluggedSensor: SensorID | null;  // ไอเทมที่ถูกนำไปติดตั้งที่แท่นบอร์ด
  currentView: 'idle' | 'success' | 'error'; // หน้าต่าง Overlay ผลลัพธ์การ Run วงจร
}
